# Multi-Agent Workflow

## Standard Build Flow

1. Orchestrator receives request
2. Planning Agent creates project plan
3. Architecture Agent defines structure
4. Database Agent designs data layer
5. Backend Agent builds API/services
6. Frontend Agent builds UI
7. Docs Agent records setup and decisions
8. Verification Agent checks everything
9. Cleanup Agent standardizes files

## Rule

No task is complete until:
- files are organized
- docs are updated
- environment is verified
- errors are logged or fixed
