# Dev Verification Checklist

Run after any major change.

## Environment
- [ ] correct Node/Python version
- [ ] env variables loaded
- [ ] dependencies installed

## Database
- [ ] DB running
- [ ] connection valid
- [ ] migrations applied

## Backend
- [ ] server starts
- [ ] no runtime errors

## Frontend
- [ ] dev server runs
- [ ] UI loads

## Code Quality
- [ ] lint passes
- [ ] types valid (if TS)
- [ ] tests pass (if applicable)

## Build
- [ ] production build succeeds
