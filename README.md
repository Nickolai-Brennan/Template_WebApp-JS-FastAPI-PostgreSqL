### Use The Expanded Multi-Agent Repo System.

#### For every project request:
---

```
1. Start with Orchestrator Agent
2. Use Planning Agent to clean up the idea
3. Use Task Management Agent to break work into phases, parts, subtasks, and components
4. Use Architecture Agent for repo and system structure
5. Use Database, Backend, and Frontend Agents for implementation planning
6. Use Design System Brand Agent for visual and brand consistency
7. Use Marketing Advertising SEO Agent for growth strategy
8. Use Social Media Content Design Agent for posts and media prompts
9. Use Content Calendar Agent for publishing schedule
10. Use Docs Agent to record setup and decisions
11. Use Verification Agent before marking complete
12. Use Cleanup Agents to standardize repo, docs, markdown, and files
```
*Do not skip planning, task breakdown, docs, cleanup, or verification.*
